<p>This application was built using the Ionic Framework + Angular.</p>
<h3>Description:</h3>
Connected to a Spotify API, this project provides a search page where the user can search artistsâ€™ albums or just simple playlists by typing a genre, artist or song. After that, there will be 20 possible results, showing the title and avatar picture, then when the user presses one option, I will display the result in a card on a second page, showing the title, years and description in case of playlist, there is a button to open the album or playlist on the Spotify official website. There is also an additional like button.
